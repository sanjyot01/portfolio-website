# Personal Website - Assignment 1 (A1)

## Project Overview
This is a responsive personal website built using HTML5, CSS3, and Flexbox framework for Assignment 1. The website showcases personal information, educational background, and interests while meeting all accessibility and validation requirements.

## File Structure
```
A1/
├── index.html          # Home page with profile image and introduction
├── about.html          # About page with personal info and YouTube video
├── interests.html      # Interests page with grid-like layout
├── styles.css          # Single external CSS file for all pages
├── images/            # Assets folder
│   ├── profile.svg    # Professional profile image placeholder
│   ├── linkedin-icon.svg
│   ├── github-icon.svg
│   ├── instagram-icon.svg
│   └── indeed-icon.svg
└── README.md          # This file
```
